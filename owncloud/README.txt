1. Use "sudo" to run owncloud-install.sh and owncloud-uninstall.sh.
To install owncloud:
Run "sudo ./owncloud-install.sh"

To uninstall owncloud:
Run "sudo ./owncloud-uninstall.sh"

2. Access owncloud via "http://localhost:8080".

3. Default authentication for owncloud:
Username: admin
Password: admin

4. Default authentication for MariaDB:
Username: owncloud
Password: owncloud
